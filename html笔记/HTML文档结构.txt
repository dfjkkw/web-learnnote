**HTML初始结构：
<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title>页面结构分析</title>
	</head>
	<body>
		向这里添加内容
	</body>
</html>

1.开头
<!DOCTYPE html> 

2.网页头部
<!--head标签代表网页头部-->
<head>  
</head>

3.描述性标签
<!--meta描述性标签，它用来描述我们网站的一些信息-->
<!--meta一般用来做SEO-->
<meta charset="utf-8"> 

4.网页标题
<title>网页标题</title> 

5.网页主体
<!--body标签代表网页主体-->
<body>向这里添加内容</body>

6.标题标签
<h1>我的第一个标题</h1>

7.段落标签
<p>我的第一个段落。</p> 

8.结尾加/
比如：</body>、</html>

**页面结构分析

1.header
标题头部区域的内容（用于页面或页面中的一块区域）

2.footer
标记脚部区域的内容（用于整个页面或页面的一块区域）
3 section
Web页面中的一块独立区域
article：独立的文章内容
aside：相关内容或应用（常用于侧边栏）
nav:导航类辅助内容


4.代码示例

页面结构：
<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title>页面结构分析</title>
	</head>
	<body>
		<header>
			<h2>网页头部</h2>
		</header>
		<section>
			<h2>网页主题</h2>
		</section>
		<footer>
			<h2>网页脚部</h2>
		</footer>
	</body>
</html>

nav标记：
<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title>nav标记</title>
	</head>
	<body>
		<header>
			<ul>
				<li><a href="#">学院首页</a></li>
				<li><a href="#">学院概况</a></li>

				<li><a href="#">招生就业</a></li> 
			</ul> 
		</header>
	</body>
</html>

article标记和section标记：
<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title>article标记和section标记</title>
	</head>
	<body>
		<article>
			<header>
				<h2>相思</h2>
				<h4>唐代：王维</h4></header>
				<section>
					红豆生南国，春来发几枝。<br>
					愿君多采撷，此物最相思。<br>
				</section>
				<aside>
					<h5>唐诗三百首，咏物，抒情，思念</h5>
				</aside>
		</article>
		<hr>
		<article>
			<header>
				<h3>译文及注释</h3>
			</header>
			<section>
				<header>
					<h4>译文:</h4>
				</header>
				<section>
					鲜红浑圆的红豆，生长在阳光明媚的南方，春暖花开的季节，不知又生出多少？	<br>
					希望思念的人儿多多采集，小小红豆引人相思。
				</section>
			</section>
			<section>
				<header>
					<h4>注释:</h4>
				</header>
				<section>⑴相思：题一作“相思子”，又作“江上赠李龟年”。<br>
					⑵红豆：又名相思子，一种生在江南地区的植物，结出的籽像豌豆而稍扁，呈鲜红色。<br>
					⑶“春来”句：一作“秋来发故枝”。<br>
					⑷“愿君”句：一作“劝君休采撷”。采撷（xié）：采摘。
					⑸相思：想念。<br>
				</section>
			</section>
			<aside>
				<h5>参考资料：<br>
				1、邓安生 等．王维诗选译．成都：巴蜀书社，1990：214-215</h5>
			</aside>
		</article>
		<hr>
		<article>
			<header>
				<h3>创作背景</h3>
			</header>
			<section>
				此诗一作《江上赠李龟年》，可见为怀念友人之作。据载，天宝末年安史之乱时，李龟年流落江南曾演唱此诗，可证此诗为天宝年间所作。
			</section>
			<aside>
				<h5>参考资料：<br>
				1、邓安生 等．王维诗选译．成都：巴蜀书社，1990：214-215</h5>
			</aside>
		</article>
		<aside>
			<h4>
				<i>来源：古诗文网http://so.gushiwen.org/view_5773.aspx</i>
			</h4>
		</aside>
	</body>
</html>

footer标记：

**什么是CSS？
CSS是Cascading Style Sheets（层叠样式表）的缩写，它是一种对Web文档添加样式的简单机制，是一种表现HTML或XML等文件外观样式的计算机语言，是一种网页排版和布局设计的技术。
纯CSS布局与结构式HTML相结合能帮助设计师将表现与结构分离，和传统的表格布局相比，具有以下特点：
1. 网页载入更快
2. 降低流量费用
3. 修改设计时效率高
4. 更容易被搜索引擎收录

**CSS历史
1996年12月W3C推出了CSS规范的第一个版本CSS1.0。
1998年W3C发布了CSS2.0/2.1版本，这也是至今流行最广并且主流浏览器都采用的标准。
2001年5月W3C就着手开发CSS第三版规范，CSS3规范被分为若干个相互独立的模块。

**CSS样式规则
使用HTML时，需要遵从一定的规范。CSS亦如此，要想熟练地使用CSS对网页进行修饰，首先需要了解CSS样式规则。
基本格式
选择器{ 
        属性1:属性值1; 
        属性2:属性值2;  
        属性3:属性值1; 
}

**CSS代码结构中的几个特点
CSS样式中的选择器严格区分大小写，属性和值不区分大小写，按照书写习惯一般将“选择器、属性和值”都采用小写的方式。
多个属性之间必须用英文状态下的分号隔开，最后一个属性后的分号可以省略，但是，为了便于增加新样式最好保留。
如果属性的值由多个单词组成且中间包含空格，则必须为这个属性值加上英文状态下的引号。
在编写CSS代码时，为了提高代码的可读性，通常会加上CSS注释。
在CSS代码中空格是不被解析的，花括号以及分号前后的空格可有可无。
